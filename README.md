## Install

```bash
npm install --save-dev @ianwremmel/eslint-config-base
(
  export PKG=@ianwremmel/eslint-config-base;
  npm info "$PKG@latest" peerDependencies --json | command sed 's/[\{\},]//g ; s/: /@/g' | xargs npm install --save-dev "$PKG@latest"
)
```
